
# SPIFLASH-OTA
SPI FLASH OTA by ESP8266 in Arduino IDE

----
#### TESTED on: W25Q64, GD25Q32C

|If use ESP8266 Pin |SPIFLASH  |if use NodeMCU Pin |
| :-----: |:-----:| :-----:|
|15|/CS-Pin1|D8|
|14|CLK-Pin6|D5|
|13|DI-Pin5|D7|
|12|DO-Pin2|D6|
|Vcc|Vcc-Pin8|3.3V|
|Vcc|/HOLD-pin7|3.3V|
|Vcc|/WP-pin3|3.3V|
|GND|GND-pin4|GND|
